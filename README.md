# Análisis de Ventas - UTN

## Descripción
Trabajo Práctico de Organización Empresarial - Tecnicatura Universitaria en Programación (UTN).

## Integrantes
- Martín Milich - Rol: Hugo (Líder), Paco (Desarrollador) y Luis (Revisor QA)

## Escenario elegido
Escenario B - Análisis de Ventas de una Pequeña Empresa

## Dataset utilizado
Dataset simulado de ventas comerciales con registros de producto, cantidad vendida, precio y fecha de venta.

## Estructura del proyecto
- /datos - Archivos de datos en formato CSV
- /scripts - Scripts de análisis en Python
- /resultados - Gráficos y resultados generados

## Instrucciones para ejecutar
1. Abrir Google Colab
2. Clonar el repositorio
3. Ejecutar el script analisis_datos.py desde la carpeta /scripts
